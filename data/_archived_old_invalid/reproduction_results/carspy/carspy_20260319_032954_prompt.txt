# CARSpy: Coherent Anti-Stokes Raman Spectroscopy Simulation and Thermometry

## Abstract
CARSpy is a Python library for simulating and fitting Coherent Anti-Stokes Raman Spectroscopy (CARS) spectra, with a focus on N2 thermometry. The library implements a physics-based forward model that includes G-matrix relaxation, Kataoka convolution, and instrument slit functions to accurately model CARS spectra. Temperature and other instrument parameters are retrieved by non-linear least-squares fitting (Levenberg-Marquardt) of the modeled spectrum to measured data.

## 1. Introduction
Coherent Anti-Stokes Raman Spectroscopy (CARS) is a powerful non-linear optical technique used for thermometry in combustion environments. The temperature of a gas can be determined by fitting the shape of the CARS spectrum, which is sensitive to the rotational-vibrational population distribution of molecules (typically N2).

The inverse problem involves minimizing the residual between a measured CARS intensity spectrum and a physics-based forward model to retrieve gas temperature and instrument parameters.

## 2. Algorithm: CARS-N2-Thermometry-Inversion

### 2.1 Objective Function
The optimization minimizes:
$$\hat{\theta} = \arg\min_{\theta} \sum_{i=1}^{N} (I_{meas}(\nu_i) - F(\nu_i; \theta))^2$$

Where F is the forward model `cars_expt_synth` and θ is the parameter set. The data fidelity term is L2 norm (sum of squared residuals). No regularization is used.

## 3. Forward Model (Algorithm Steps)

The forward model computes the theoretical CARS spectrum through these steps:

### Step 1: Coordinate Transformation
Transform experimental wavenumbers: ν' = ν_expt × nu_stretch + nu_shift.
Generate fine grid ν_f by padding and reducing step size by factor ref_fac=10.

### Step 2: Energy Levels & Populations
Calculate term values G(v), F_v(J) using Dunham expansion coefficients.
Calculate population difference ΔN using Boltzmann distribution at temperature T.

### Step 3: Relaxation Matrix (MEG Model)
Calculate off-diagonal rates γ_ji using the Modified Exponential Gap (MEG) model.
Diagonal elements: γ_ii = -Σ(j≠i) γ_ji.

### Step 4: Resonant Susceptibility (G-Matrix)
Construct kernel K = diag(ν_Raman) + iΓ.
Eigendecomposition: K = VΛV⁻¹.
Compute: χ_rs(ν) = Σ_j (d·V)_j (V⁻¹·ΔN·d)_j / (-ν + λ_j)

### Step 5: Total Susceptibility
χ_total = N_dens × x_mol × χ_rs + χ_nrs
I_raw = |χ_total|²

### Step 6: Kataoka Convolution
Generate pump line profile (Gaussian/Lorentzian).
Convolve susceptibility and intensity with pump line.
Combine: I_as = 0.5 × (I_conv1 + Δν × |χ_conv|²)

### Step 7: Instrument Slit Convolution
Generate slit function S(ν) (Asymmetric Gaussian/Voigt).
I_final = I_as * S(ν)

### Step 8: Downsampling & Normalization
Interpolate I_final from ν_f back to ν_expt.
Normalize: I_model = I_down / max(I_down)

### Step 9: Residual
r = I_measured - I_model
